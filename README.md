## Usage

1. Copy the `moonlit-harvest-theme.json` file from this repo into `~/.config/zed/themes`.
2. Open the theme selector palette (<kbd>Cmd</kbd>+<kbd>K</kbd> <kbd>Cmd</kbd>+<kbd>T</kbd>)
3. Select the `Moonlit Harvest` theme.
